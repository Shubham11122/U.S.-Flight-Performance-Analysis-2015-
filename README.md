# U.S. Flight Performance Analysis (2015)

## Project Overview

This project analyzes over 5.8 million U.S. domestic flight records to identify patterns in flight delays, cancellations, airline performance, and airport operations.

The objective was to transform raw aviation data into actionable business insights using Python, SQL Server, and Power BI.

---

## Tools Used

* Python (Pandas)
* SQL Server
* Power BI
* DAX

---

## Project Workflow

### 1. Data Cleaning (Python)

* Standardized column names
* Handled missing values
* Validated data quality
* Created derived columns:

  * Flight Date
  * Month Name
  * Day Name
* Performed duplicate checks

### 2. Data Analysis (SQL Server)

* KPI generation
* Airline performance analysis
* Airport performance analysis
* Delay trend analysis
* Cancellation analysis
* Delay cause analysis

### 3. Dashboard Development (Power BI)

Created three interactive dashboards:

#### Dashboard 1: U.S. Flight Performance Overview

* Total Flights
* Average Arrival Delay
* Average Departure Delay
* Cancellation Rate
* Monthly Delay Trends
* Cancellation Analysis

#### Dashboard 2: Airline Performance Analysis

* Best and Worst Airlines
* Flight Volume by Airline
* Average Delay by Airline
* Cancellation Rate by Airline
* On-Time Performance

#### Dashboard 3: Airport Performance Analysis

* Airport Traffic Analysis
* Most Delayed Airports
* Airport Performance Map
* Delay Cause Analysis
* Airport Performance Scorecard

---

## Key Findings

* Over 5.8 million flights were analyzed.
* Overall cancellation rate was approximately 1.54%.
* Departure delays were significantly higher than arrival delays.
* Weather and airline-related factors were major contributors to delays.
* Certain airports consistently experienced higher delay rates.
* Significant differences were observed in airline on-time performance.

---

## Business Recommendations

* Improve scheduling for high-delay airlines.
* Strengthen weather contingency planning.
* Optimize aircraft rotation and maintenance planning.
* Increase operational efficiency at high-traffic airports.
* Implement real-time delay monitoring systems.
